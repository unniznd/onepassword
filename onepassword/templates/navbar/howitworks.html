Using the same passwords on different websites is not safe. But remembering different passwords is a herculean task. However, using a password manager is a good solution but not the best solution. Since passwords as kept in servers.

We OnePassword give you the best solution. The user only needs to remember a master password and an easily recallable keyword to generate different copies of the master password. The keyword can be hostname with different cases.

For example,
master password = password123
keyword for Spotify is spotify
Generated the password bc9d1c3a9a02a4e1.

This keyword adds additional security to the password. None of the data are recorded by OnePassword.
